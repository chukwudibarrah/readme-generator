# README Generator

## Description

The goal was to understand how the inquirer package works alongside node js and JavaScript
The application helps users easily and quickly create a professional
The main challenge it solves is helping creators quickly create professional README files without thinking too much
How inquirer works and displaying output in the terminal as well as being able to create files using the fs (file system) module for node
